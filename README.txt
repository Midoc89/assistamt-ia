# Assistant IA - Déploiement Render

## Étapes pour Render

1. Uploade ce projet sur un dépôt GitHub
2. Va sur https://render.com et crée un nouveau Web Service
3. Connecte ton GitHub et sélectionne ce dépôt
4. Configure :
   - Runtime : Node
   - Start Command : npm start
   - Environment Variable : OPENAI_API_KEY = ta clé OpenAI
   - Root Directory : /

5. Clique sur Deploy

Ton assistant sera disponible en ligne !
